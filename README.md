# Яндекс трекер развития - фронтенд

# Требования
Node 18+  
npm 7+

# Установка
 1. Склонировать репозиторий
 2. Установить зависимоcти командой: 
   
    ```npm install``` или ```yarn install```

# Скрипты

- Дев сервер  
```npm run dev```

- Превью   
```npm run preview```

- Билд  
```npm run build```

- Форматирование prettier  
```npm run prettier```

- Запуск eslint  
```npm run lint```

# Технологии

- Typescript
- React 18
- Redux-toolkit
- React router
- Vite@4
- Sass
